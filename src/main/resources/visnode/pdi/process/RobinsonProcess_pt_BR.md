# Robinson

Robson propôs um conjunto de oito máscaras onde a magnitude é calculada de
maneira semelhante a realizada por Kirsh, ou seja, utilizando o valor máximo entre as
oito máscaras (PEDRINI; SCHWARTZ, 2008).
